# Tableau Dashboard Portfolio

A collection of interactive dashboards created using Tableau Public. This repository showcases data visualization, analytics, KPI tracking, and business intelligence projects built using real-world datasets across multiple domains including gaming, healthcare, road accidents, sales, and more.

## Tools & Technologies

* Tableau Public
* Data Visualization
* Interactive Dashboards
* Calculated Fields
* Filters & Parameters

## Features

* Interactive Charts
* KPI Tracking
* Dynamic Filters
* Data Storytelling
* Dashboard Design
* Visual Analytics

## Purpose

This repository highlights Tableau dashboard development skills, interactive reporting, and business intelligence techniques through visually engaging and analytical dashboards.
